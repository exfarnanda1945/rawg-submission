# rawg-submission
Become An Android Developer RAWG Submission

# Used Dependency:
- retrofit
- logging interceptor
- glide
- viewpager
- circle-indicator
- card view

# Features
- Image Slider / Carousel
- RecyclerView Horizontal
- Fragment for Search Game
- Fragment for Detail Game
- etc
